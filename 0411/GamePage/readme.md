### 게임을 시작하는 시작 페이지를 작성.
#### 메인페이지는 하나의 이미지와 3개의 버튼으로 구성.
#### 각각의 버튼을 누르면 해당되는 페이즈로 이동.
#### 이동한 페이지에서 BACK키를 누르면 메인 페이지로 되돌아옴.
