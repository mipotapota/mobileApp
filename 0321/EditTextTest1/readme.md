# 사용자로부터 아이디, 패스워드, 전화번호를 입력받을 수 있는 안드로이드 앱 만들기. 적절한 inputType을 사용.

### android:inputType="text" <- 한 줄만 입력 가능. android:inputType="textPassword" <- 입력한 내용이 보이지 않음. android:inputType"phone" <- 전화번호만 입력 가능.
