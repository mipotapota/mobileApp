# 상수 visible, invisible, gone을 이용하여 위젯의 표시 여부 설정.
## android:visibility="visible" <- 화면에 보이게 한다. 디폴트 값 android:visibility="invisible" <- 표시되지 않는다. 그러나 배치에서 공간을 차지한다. android:visibility="gone" -> 완전히 숨겨진다.
